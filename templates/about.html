About the Project

The Eye Risk Prediction System is a data-driven web application designed to assess
an individual's risk of developing eye-related health conditions using machine learning techniques.
By analyzing key physiological and lifestyle parameters
such as age, eye pressure, retina thickness, vision clarity, sleep patterns, cholesterol, and
blood sugar levels, the system provides real-time risk predictions categorized as Low, Moderate, or High.
Built with a user-friendly interface and powered by predictive models
like Random Forest or Logistic Regression, the platform aims to support early detection and preventive eye care.
It empowers healthcare professionals with valuable insights and
enables individuals to take proactive measures toward maintaining optimal eye health.

How It Works

Users input key health parameters such as age, eye pressure, retina thickness, and lifestyle metrics into the system.
The application preprocesses the data and uses a trained machine learning model to analyze patterns and predict eye health risk levels.
Based on the prediction (Low, Moderate, or High), it displays the risk level and visual insights through interactive graphs for better understanding and early action.
